Comments

Comments allow us to write things in our code that is purely for notation (like writing notes in your math). Anything in the same line that comes after "//" will not be used while your code is running.

We can use comments for things like add a TODO message so you can come back to add something later, or we can use it to help explain how our code works.

Flow Control

Flow control allows us to establish branching choices in our logic model using boolean logic. You might say something along the lines of "if x is true then do y, or else do z". It allows us to create flexible code that chooses the appropriate path to take in order to properly complete the task. You can read more about flow control here.
